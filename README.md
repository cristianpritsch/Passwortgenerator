# 🔐 Passwortgenerator (Python)

Ein einfacher Passwortgenerator, der mit Python erstellt wurde.

## Funktionen

- Passwortlänge frei wählbar
- Verwendung von Groß- und Kleinbuchstaben
- Zahlen
- Sonderzeichen
- Einfache Benutzereingabe über die Konsole

## Verwendete Technologien

- Python 3

## Beispiel

```text
=== Passwortgenerator ===

Wie lang soll das Passwort sein? (mindestens 8 Zeichen): 12

Dein Passwort lautet:

B#8mLp!2Qz@7
```

Dieses Projekt wurde erstellt, um meine Python-Grundkenntnisse zu vertiefen und den Umgang mit Benutzereingaben, Schleifen und Zufallswerten zu üben.
